## Data collections (DCDs)

The register contains 5 data collection definitions:

1. **Hospitalisation** — hospitalisation without PCI or FFR
2. **Hospitalisation with PCI** — percutaneous coronary intervention
3. **Hospitalisation with FFR** — fractional flow reserve measurement
4. **Hospitalisation with FFR and PCI** — combined FFR and PCI
5. **Follow-up after PCI** — post-PCI follow-up

## Data submission

- Manual registration via HD4DP v2
- CSV upload
- S2S API

## DCD structure

The Excel specification file contains field names, technical keys, data types, code values (including references to general HD4DP2 code lists), validation rules, help texts, error messages, and translations (NL/FR/EN).
